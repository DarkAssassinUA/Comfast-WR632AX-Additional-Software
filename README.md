# Дополнительное ПО для Comfast WR632AX

Этот репозиторий содержит дополнительные скрипты и файлы для роутера Comfast CF-WR632AX на базе OpenWrt.

## Установка и запуск ufan.sh

Скрипт `ufan.sh` представляет собой интерактивный установщик службы автоматического управления кулером `uFan` с дополнительной поддержкой мониторинга вентилятора (Fan monitoring для Luci).

### Быстрая установка

Подключитесь к роутеру по SSH и выполните следующую команду для загрузки и запуска установщика:

```bash
wget --no-check-certificate -qO ufan.sh "https://raw.githubusercontent.com/DarkAssassinUA/Comfast-WR632AX-Additional-Software/main/ufan.sh" && chmod +x ufan.sh && ./ufan.sh
```

### Возможности установщика:
1. **Интерактивное меню** для удобного управления.
2. **Проверка статуса установки** службы `uFan` и модуля мониторинга.
3. **Опциональная установка мониторинга вентилятора** (Fan monitoring через collectd для веб-интерфейса Luci) с автоматической установкой зависимостей.
4. **Безопасное удаление** всех компонентов одной командой.
5. **Автоматическое обновление** скрипта при запуске напрямую из репозитория GitHub.
6. **Интеграция с sysupgrade** для сохранения установленного ПО при обновлении прошивки.

## Установка и запуск theme-proton.sh

Скрипт `theme-proton.sh` представляет собой интерактивный установщик современной темы оформления **Proton2025** для веб-интерфейса LuCI роутера.

### Быстрая установка

Подключитесь к роутеру по SSH и выполните следующую команду для загрузки и запуска установщика темы:

```bash
wget --no-check-certificate -qO theme-proton.sh "https://raw.githubusercontent.com/DarkAssassinUA/Comfast-WR632AX-Additional-Software/main/theme-proton.sh" && chmod +x theme-proton.sh && ./theme-proton.sh
```

### Возможности установщика:
1. **Интерактивное меню** с выводом текущего статуса и установленной версии темы.
2. **Автоматическое скачивание и установка** последней актуальной версии темы с GitHub-релизов автора (`luci-theme-proton2025`).
3. **Поддержка любых пакетных менеджеров**: корректно работает как со старым `opkg` (.ipk), так и с новым `apk` (.apk) в последних сборках OpenWrt.
4. **Удаление темы** одной командой с последующей очисткой кэша LuCI.
5. **Автоматическая фоновая проверка обновлений** установщика и самой темы на GitHub при каждом запуске.
